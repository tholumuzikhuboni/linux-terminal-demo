Advanced Linux Commands

Learn about the most commonly used Linux commands for advanced system operations.

1. `ls` - List Directory Contents

The `ls` command is used to list the files and directories in the current working directory. You can also pass options like `-l` to get detailed information, or `-a` to show hidden files.

2. `cd` - Change Directory

The `cd` command allows you to change the current working directory. For example, `cd /home/user` changes the directory to `/home/user`.

3. `cp` - Copy Files or Directories

The `cp` command is used to copy files or directories from one location to another. For example, `cp file1.txt file2.txt` copies `file1.txt` to `file2.txt`.

4. `mv` - Move or Rename Files/Directories

The `mv` command moves or renames files and directories. For example, `mv oldname.txt newname.txt` renames a file.

5. `rm` - Remove Files or Directories

The `rm` command is used to delete files and directories. To remove directories, use `rm -r` for recursive deletion.

6. `chmod` - Change File Permissions

The `chmod` command changes the permissions of a file or directory. It allows you to control who can read, write, or execute the file.

7. `chown` - Change File Ownership

The `chown` command changes the ownership of a file or directory. For example, `chown user:group file.txt` changes the ownership of `file.txt` to `user` and the group to `group`.

8. `ps` - View Running Processes

The `ps` command displays information about running processes on the system. You can use options like `-aux` to view all processes.

9. `top` - Task Manager

The `top` command shows a real-time, dynamic view of the system's resource usage, including CPU, memory, and running processes.

10. `grep` - Search for Patterns in Files

The `grep` command searches for a specific pattern or string in files. For example, `grep "searchterm" file.txt` searches for `searchterm` in `file.txt`.

11. `df` - Disk Space Usage

The `df` command shows the amount of disk space used and available on your file system. Use the `-h` option to display human-readable sizes.

12. `du` - Disk Usage of Files

The `du` command displays the disk usage of files and directories. Use `du -sh` for a summary of a directory's size.

13. `sudo` - Execute Commands as Superuser

The `sudo` command allows you to execute a command with superuser privileges. This is essential for administrative tasks like installing software.

14. `wget` - Download Files from the Internet

The `wget` command is used to download files from the web. For example, `wget http://example.com/file.txt` will download `file.txt` from the specified URL.

15. `tar` - Archive Files

The `tar` command is used to create or extract compressed archives. For example, `tar -czvf archive.tar.gz /directory` creates a compressed archive of a directory.
