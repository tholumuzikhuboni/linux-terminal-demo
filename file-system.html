Linux File System Commands

Learn about the essential file system commands in Linux used for navigating and managing files and directories.

1. `pwd` - Print Working Directory

The `pwd` command prints the current working directory, allowing you to see where you are within the file system.

2. `cd` - Change Directory

The `cd` command allows you to change the current working directory. For example, `cd /home/user` changes the directory to `/home/user`.

3. `ls` - List Directory Contents

The `ls` command is used to list the files and directories in the current directory. Options like `-l` provide detailed information.

4. `mkdir` - Make Directories

The `mkdir` command creates new directories. For example, `mkdir mydir` creates a directory named `mydir`.

5. `rmdir` - Remove Empty Directory

The `rmdir` command is used to remove empty directories. If the directory contains files, you must delete them first.

6. `rm` - Remove Files and Directories

The `rm` command removes files and directories. Use `rm -r` to remove directories recursively.

7. `cp` - Copy Files and Directories

The `cp` command copies files or directories from one location to another. Use the `-r` flag for directories.

8. `mv` - Move or Rename Files

The `mv` command moves files and directories or renames them. For example, `mv file.txt newname.txt` renames the file.

9. `find` - Search for Files

The `find` command searches for files and directories. For example, `find /home/user -name "*.txt"` finds all `.txt` files in the specified directory.

10. `touch` - Create Empty Files

The `touch` command creates empty files or updates the timestamp of existing files. For example, `touch newfile.txt` creates a new empty file.

11. `ln` - Create Hard and Symbolic Links

The `ln` command creates hard and symbolic links between files. Use `ln -s` for creating symbolic links.

12. `df` - Disk Space Usage

The `df` command reports file system disk space usage, displaying available and used space. Use the `-h` flag for human-readable formats.

13. `du` - Disk Usage of Files and Directories

The `du` command shows disk usage of files and directories. Use `du -sh` to get a summary of the total space used by a directory.

14. `chmod` - Change File Permissions

The `chmod` command changes the file permissions. For example, `chmod 755 file.txt` grants execute permissions to the owner.

15. `chown` - Change File Ownership

The `chown` command changes file ownership. For example, `chown user:group file.txt` changes the ownership of the file to the specified user and group.
